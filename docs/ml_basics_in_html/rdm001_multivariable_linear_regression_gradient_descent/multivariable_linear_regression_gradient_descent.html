Define the latest squares cost function¶

Lets say we have sample data set:¶

$$ \begin{align*} & \ (x_1^{(1)}, x_2^{(1)}, \dots, x_m^{(1)}, y^{(1)}), (x_1^{(2)}, x_2^{(2)}, \dots, x_m^{(2)}, y^{(2)}), \dots, (x_1^{(n)}, x_2^{(n)}, \dots, x_m^{(n)}, y^{(n)}) \\ & \ x_i^{(j)} \ \Big( \begin{aligned} i = 1, 2, \dots, m \\ j = 1, 2, \dots, n \end{aligned} \Big) \ \text{represents the value of the feature } x_i \ \text{of the } j^{th} \ \text{sample record} \\ & \ y^{(i)} \ \big( \begin{aligned} i = 1, 2, \dots, n \end{aligned} \Big) \ \text{represents the target value of the } i^{th} \ \text{sample record} \end{align*} $$

And we want to find out the fitting equation:¶

$$ y_\theta(x_1, x_2, \dots, x_m) = \theta_0 + \theta_1x_1 + \dots + \theta_mx_m $$

to minimize the quadratic sum of each sample data's fitting error $\varepsilon_i$, that is: $\sum\limits_{i=1}^n\varepsilon_i^2$, which the $\varepsilon_i$ can be represented in the below way:

$$ \begin{cases} y^{(1)} = \theta_0 + \theta_1x_1^{(1)} + \dots + \theta_mx_m^{(1)} + \varepsilon_1 \\ y^{(2)} = \theta_0 + \theta_1x_1^{(2)} + \dots + \theta_mx_m^{(2)} + \varepsilon_2 \\ \vdots \\ y^{(n)} = \theta_0 + \theta_1x_1^{(n)} + \dots + \theta_mx_m^{(n)} + \varepsilon_n \end{cases} $$

Then we can calculate and define the cost funciton:¶

$$ \begin{align*} C(\theta_0, \theta_1, \dots, \theta_m) \ &= \dfrac{1}{2n}\sum\limits_{i=1}^n\varepsilon_i^2 \\ &= \dfrac{1}{2n}\big\{ \ [(\theta_0 + \theta_1x_1^{(1)} + \dots + \theta_mx_m^{(1)}) - y^{(1)}]^2 + \ [(\theta_0 + \theta_1x_1^{(2)} + \dots + \theta_mx_m^{(2)}) - y^{(2)}]^2 + \ \dots + \ [(\theta_0 + \theta_1x_1^{(i)} + \dots + \theta_mx_m^{(i)}) - y^{(n)}]^2 \ \big\} \\ &= \dfrac{1}{2n}\sum\limits_{i=1}^n[y_\theta(x_1^{(i)}, x_2^{(i)}, \dots, x_m^{(i)}) - y^{(i)}]^2 \\ &= \dfrac{1}{2n}\sum\limits_{i=1}^n\Bigg[ \ \begin{pmatrix} 1 & x_1^{(1)} & x_2^{(1)} & \dots & x_m^{(1)} \\ 1 & x_1^{(2)} & x_2^{(2)} & \dots & x_m^{(2)} \\ \vdots \\ 1 & x_1^{(n)} & x_2^{(n)} & \dots & x_m^{(n)} \end{pmatrix} \begin{pmatrix} \theta_0 \\ \theta_1 \\ \vdots \\ \theta_m \end{pmatrix} - \ \begin{pmatrix} y^{(1)} \\ y^{(2)} \\ \vdots \\ y^{(n)} \end{pmatrix} \Bigg]^2 \end{align*} $$

Then we can define the gradient descent algorithm as below:¶

$ \text{repeat until convergence }\Bigg\{ $

$$ \begin{align*} \theta_0 &= \theta_0 - \alpha\dfrac{\partial}{\partial\theta_0}C(\theta_0, \theta_1, \dots, \theta_m) \ = \theta_0 - \alpha\dfrac{1}{n}\sum\limits_{i=1}^n[y_\theta(x_1^{(i)}, x_2^{(i)}, \dots, x_m^{(i)}) - y^{(i)}] \ = \theta_0 - \alpha\dfrac{1}{n}\sum\limits_{i=1}^n[y_\theta(x_1^{(i)}, x_2^{(i)}, \dots, x_m^{(i)}) - y^{(i)}]x_0^{(i)} \:\: \big(x_0^{(i)} = 1\big) \\ \theta_1 &= \theta_1 - \alpha\dfrac{\partial}{\partial\theta_1}C(\theta_0, \theta_1, \dots, \theta_m) \ = \theta_1 - \alpha\dfrac{1}{n}\sum\limits_{i=1}^n[y_\theta(x_1^{(i)}, x_2^{(i)}, \dots, x_m^{(i)}) - y^{(i)}]x_1^{(i)} \\ \vdots \\ \theta_m &= \theta_m - \alpha\dfrac{\partial}{\partial\theta_m}C(\theta_0, \theta_1, \dots, \theta_m) \ = \theta_m - \alpha\dfrac{1}{n}\sum\limits_{i=1}^n[y_\theta(x_1^{(i)}, x_2^{(i)}, \dots, x_m^{(i)}) - y^{(i)}]x_m^{(i)} \end{align*} $$

$ \Bigg\} $

So we got three points from the above calculations¶

  1. Cost function.
  2. We need to insert one all 1 column to all the sample data to make the matrix calculations can be performed correctly.
  3. The gradient descent algorithm.

# Define the latest squares cost function.
def computeCost(X, y, theta):
    error = np.power((X * theta.T) - y, 2)
    # The '1/2' is just for calculation convenience, since we know we will use ‘gradient descent’ algorithm,
    # and the cost function is one second derived function which after derivative there will be one '2' there.
    return np.sum(error) / (2 * len(X))

def gradientDescent(X, y, theta, alpha, iters):
    numOfTheta = int(theta.shape[1])
    cost = np.zeros(iters)

    for i in range(iters):
        error = (X * theta.T) - y

        # Loop to update each θ.
        for j in range(numOfTheta):
            term = np.multiply(error, X[:, j])
            theta[0, j] = theta[0, j] - alpha / len(X) * np.sum(term)

        cost[i] = computeCost(X, y, theta)

    return theta, cost
